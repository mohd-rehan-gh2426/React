# **Multidimensional Arrays in JavaScript**

## **1️⃣ What is a multidimensional array?**

* A **multidimensional array** is simply an **array inside another array**.
* Think of it as “grouping arrays together.”
* Despite the scary name, it’s **very simple**.

**Example of an empty multidimensional array:**

```javascript
let multiArr = [[]]; // One array inside another array
console.log(multiArr); // [[]]
console.log(multiArr.length); // 1 → outer array has 1 element
```

---

## **2️⃣ Adding elements to a multidimensional array**

You can store **anything** inside an array: strings, numbers, objects, or even other arrays.

```javascript
let students = [
  ["Aadarsh", 75], // Name + percentage
  ["Akash", 90]
];

console.log(students);
/*
[
  ["Aadarsh", 75],
  ["Akash", 90]
]
*/
```

* Each element of the **outer array** is itself an **array** (inner array).
* This is useful to **group related data together**: e.g., name and score of a student.

---

## **3️⃣ Accessing elements in a multidimensional array**

* To access an inner array: use the **index of the outer array**.

```javascript
console.log(students[1]); // ["Akash", 90] → 1st index of outer array
```

* To access a **specific element inside the inner array**:

```javascript
console.log(students[1][0]); // "Akash" → first element of inner array
console.log(students[1][1]); // 90 → second element of inner array
```

* **Remember:** first bracket → outer array, second bracket → inner array.

---

## **4️⃣ Updating elements**

* You can **update an element** like a normal array:

```javascript
students[1][1] = 95; // Update Akash's score
console.log(students[1]); // ["Akash", 95]
```

* You can also add a new index:

```javascript
students[1][3] = "Passed"; // creates a 3rd index
console.log(students[1]); // ["Akash", 95, undefined, "Passed"]
```

---

## **5️⃣ Example Use Case: Tic-Tac-Toe Board**

* A multidimensional array is great for **grids or tables**, like a Tic-Tac-Toe board.

```javascript
const ticTacToe = [
  ["X", null, null], // 1st row
  [null, "O", null], // 2nd row
  ["O", null, "X"]   // 3rd row
];

console.log(ticTacToe[0]); // ["X", null, null] → first row
console.log(ticTacToe[1][1]); // "O" → middle element of 2nd row
console.log(ticTacToe[2][2]); // "X" → last element of 3rd row
```

* Accessing rows → `ticTacToe[rowIndex]`
* Accessing specific cell → `ticTacToe[rowIndex][colIndex]`

---

## **6️⃣ Key Points to Remember**

1. **Multidimensional arrays = array of arrays**.
2. Can store **anything** inside: numbers, strings, objects, or even more arrays.
3. **Accessing elements** requires **nested brackets**.
4. Can go as **deep as you want**:

```javascript
let arr = [[[[["Deep"]]]]];
console.log(arr[0][0][0][0][0]); // "Deep"
```

5. Can be used to **group related data** like student name & score, or create **grids/tables**.
6. **Outer array length** = number of groups/rows.
   **Inner array length** = number of items per group/row.

---

## **7️⃣ Practical Example: Student Data**

```javascript
let studentList = [
  ["Aadarsh", 75],
  ["Akash", 90],
  ["Rayhaan", 99]
];

console.log(studentList[0]);    // ["Aadarsh", 75]
console.log(studentList[0][0]); // "Aadarsh"
console.log(studentList[2][1]); // 99

// Update score
studentList[2][1] = 100;
console.log(studentList[2]);    // ["Rayhaan", 100]
```

* Each student is represented as an inner array: `[name, score]`.
* Easy to **access**, **update**, or **iterate**.

---

## **8️⃣ Bonus: Multidimensional Array Tips**

* You don’t need to call them “multidimensional” to work with them; it’s just arrays inside arrays.
* You can **mix types** in the inner arrays, but it’s better to be consistent.
* Arrays can go **many levels deep**, but readability becomes difficult beyond 2-3 levels.

---

✅ **Summary:**

* Multidimensional arrays = array of arrays.
* Access elements with multiple `[ ]`.
* Useful for **grouped data** or **grids** like tic-tac-toe.
* Can go multiple levels deep.
* Updating/adding elements works same as normal arrays.
