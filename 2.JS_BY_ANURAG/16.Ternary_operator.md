```md
# JavaScript Ternary Operator (Complete Notes)

---

## 1. What is the Ternary Operator?

The **ternary operator** is a **short and concise alternative to `if-else`**.

- Used for **decision making**
- Returns **one value**
- Written in **one line**
- Very common in real-world JavaScript (frontend, backend, React)

---

## 2. Why is it called Ternary?

Because it works with **three parts**:

```

condition ? value_if_true : value_if_false

````

- `condition` â†’ evaluated to true or false
- `?` â†’ if
- `:` â†’ else

## 3. Basic Syntax

```js

````

### Rule

* If condition is **true** â†’ `expression1` is returned
* If condition is **false** â†’ `expression2` is returned

## 7. Real-World Example: Gender (He / She)

### Problem

Display:

* `"He is a college student"` for male
* `"She is a college student"` for female

---

### Step 1: Gender Variable

```js
const gender = "F";
```

---

### Step 2: Template Literal with Ternary

```js

  `${gender === "F" ? "She" : "He"} is a college student`;
```

âœ” Output:

```

```

---

## 8. Why Template Literals?

Ternary returns a value, and template literals allow inserting values inside strings.

```js
`${condition ? value1 : value2}`
```

Used when part of a sentence depends on a condition.

---

## 9. Case Sensitivity Issue

JavaScript is **case-sensitive**.

```js
"F" !== "f"
```

So this fails:

```js
gender === "F"
```

If gender is `"f"`.

---

## 10. Solution: Normalize Using `toLowerCase()`

```js
const userMessage =
  `${gender.toLowerCase() === "f" ? "She" : "He"} is a college student`;
```

âœ” Works for `"F"` and `"f"`
âœ” Best practice in real projects

## 11. Truthy and Falsy Values

Ternary **does NOT require only true/false**.

It works on **truthy** and **falsy** values.

---

### Falsy Values in JavaScript

```js
false
0
""
null
undefined
NaN
```

Everything else is **truthy**.

---

### Examples

```js
5 ? "She" : "He"     // She (truthy)
0 ? "She" : "He"     // He (falsy)
"" ? "She" : "He"    // He (falsy)
" " ? "She" : "He"   // She (truthy)
```

```js

  status = "Adult";

  status = "Minor";
}
```

### Ternary

