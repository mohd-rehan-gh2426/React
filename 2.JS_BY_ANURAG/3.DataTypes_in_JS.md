# JavaScript Data Types (Primitive Data Types)

## 1. What is a Data Type?

A **data type** tells JavaScript **what kind of value** you are storing or working with.

Examples:

* Numbers → `10`, `3.14`
* Text → `"hello"`
* Boolean values → `true`, `false`
* Empty values → `undefined`, `null`

JavaScript uses data types to **store, process, and operate** on values correctly.

---

## 2. Categories of Data Types in JavaScript

JavaScript has **two main categories**:

1. **Primitive Data Types** ✅
2. **Non-Primitive Data Types** (Objects, Arrays, Functions – later)

### Primitive Data Types

* Store **single values**
* Immutable (cannot be changed directly)
* Lightweight

---

## 3. Primitive Data Types Overview

JavaScript has **7 primitive data types**:

1. `Number`
2. `String`
3. `Boolean`
4. `Undefined`
5. `Null`
6. `BigInt`
7. `Symbol`

> This file focuses on the **commonly used 5 types**.

---

## 4. typeof Operator

Used to check the data type of a value.

```js
typeof 7        // "number"
typeof "hello"  // "string"
```

---

## 5. Number Data Type

### Key Points

* JavaScript has **only one number type**
* No separate `int`, `float`, or `double`

```js
7
-10
3.14
```

All return:

```js
typeof 3.14 // "number"
```

⚠ Very large numbers may lose precision.

---

## 6. String Data Type

Strings represent **text data**.

### Ways to Create Strings

```js
'Hello'
"Hello"
`Hello`
```

All are valid strings:

```js
typeof 'Hello' // "string"
```

### Empty String

```js
''
""
``
```

Still a string:

```js
typeof '' // "string"
```

---

## 7. Backticks (Template Literals)

Backticks allow **multiline strings**.

```js
`Hello
World`
```

Single and double quotes **do not support multiline strings**.

---

## 8. String to Number Conversion

### Unary Plus (`+`)

```js
+'100'      // 100
+'abc'      // NaN
```

### NaN (Not a Number)

* Indicates an invalid number
* Type is still `number`

```js
typeof NaN // "number"
```

### parseInt()

```js
parseInt('100abc') // 100
parseInt('abc100') // NaN
```

---

## 9. Number to String Conversion

```js
100 + ''   // "100"
'' + 100   // "100"
```

JavaScript prefers **string concatenation**.

---

## 10. Boolean Data Type

```js
true
false
```

```js
typeof true // "boolean"
```

### Boolean to Number

```js
+true   // 1
+false  // 0
```

Used in conditions and logic.

---

## 11. Undefined

```js
undefined
```

* Value is not assigned
* Automatically assigned by JavaScript

```js
typeof undefined // "undefined"
```

---

## 12. Null

```js
null
```

* Intentional empty value
* Assigned by programmer

```js
typeof null // "object" // JavaScript bug
```

### Interview Note

`typeof null === "object"` is a **historic JavaScript bug**.

---

## 13. Null vs Undefined Conversion

```js
+null       // 0
+undefined  // NaN
```

---
