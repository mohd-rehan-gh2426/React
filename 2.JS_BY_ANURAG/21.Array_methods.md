## 1Ô∏è‚É£ **Shift and Unshift**

### **Shift**

* `shift()` removes the **first element** from an array.
* All other elements **shift forward**, which changes their indexes.
* Example:

```javascript
let evenNumbers = [0, 2, 4, 6, 8];
console.log(evenNumbers.shift()); // 0
console.log(evenNumbers); // [2, 4, 6, 8]
```

üí° **Important:**
Removing from the front takes more work than removing from the back (`pop()`), because all indexes have to be updated.

---

### **Unshift**

* `unshift(element)` adds one or more elements at the **start of the array**.
* Example:

```javascript
evenNumbers.unshift(0);
console.log(evenNumbers); // [0, 2, 4, 6, 8]
```

* Returns the **new length** of the array:

```javascript
console.log(evenNumbers.unshift(-2)); // 6 (new length)
console.log(evenNumbers); // [-2, 0, 2, 4, 6, 8]
```

‚úÖ Shift/Unshift modify the **original array**.

## 2Ô∏è‚É£ **Concat**

* `concat()` combines **two or more arrays** without modifying the originals.
* Returns a **new array**.

```javascript
let animals = ["dog", "cat", "rat"];
let allArray = evenNumbers.concat(animals);
console.log(allArray);
// [-2, 0, 2, 4, 6, 8, "dog", "cat", "rat"]
console.log(evenNumbers); // original unchanged
console.log(animals);     // original unchanged
```

* You can **concatenate multiple arrays** at once:

```javascript
let oddNumbers = [1, 3, 5, 7, 9];
let combined = evenNumbers.concat(animals, oddNumbers);
console.log(combined);
```

## 3Ô∏è‚É£ **IndexOf**

* `indexOf(element)` gives the **first index** of the element in the array.
* Returns `-1` if the element does not exist.

```javascript
console.log(evenNumbers.indexOf(4)); // 3
console.log(evenNumbers.indexOf(10)); // -1
```

* Works for strings too:

```javascript
console.log(animals.indexOf("cat")); // 1
console.log(animals.indexOf("Cat")); // -1 (case-sensitive)
```

---

## 4Ô∏è‚É£ **Includes**

* `includes(element)` checks if an element exists in the array.
* Returns **true** or **false**.

```javascript
console.log(animals.includes("cat")); // true
console.log(animals.includes("bat")); // false
```

---

## 5Ô∏è‚É£ **Reverse**

* `reverse()` flips the **order of the elements**.
* Modifies the **original array**.

```javascript
console.log(animals); // ["dog", "cat", "rat", "lion"]
animals.reverse();
console.log(animals); // ["lion", "rat", "cat", "dog"]
```

---

## 6Ô∏è‚É£ **Sort**

* `sort()` sorts an array **alphabetically** by default.
* **Modifies the original array.**

```javascript
animals.sort();
console.log(animals); // ["cat", "dog", "lion", "rat"]
```

‚ö†Ô∏è **Caution with numbers:**
By default, `sort()` converts numbers to strings and sorts alphabetically:

```javascript
let numbers = [0, 10, 2, 4];
numbers.sort();
console.log(numbers); // [0, 10, 2, 4] (wrong numeric order)
```

* Can pass a **function** to sort numbers correctly (will learn later with functions).

* Case-sensitive:

  * `"Cat"` and `"cat"` sort differently because uppercase and lowercase have different **UTF-16 codes**.

## 7Ô∏è‚É£ **Slice vs Splice**

### **Slice**

* `slice(start, end)` **does not modify** the original array.
* Returns a **new array** from `start` index **up to but not including** `end`.
* Both `start` and `end` are optional.

```javascript
let someAnimals = ["dog", "cat", "rat", "lion", "elephant"];
let part = someAnimals.slice(1, 4);
console.log(part); // ["cat", "rat", "lion"]
console.log(someAnimals); // original unchanged
```

---

### **Splice**

* `splice(start, deleteCount, item1, item2, ...)` **modifies the original array**.
* Can **remove** and **add elements**.

```javascript
let animals = ["dog", "cat", "rat", "lion", "elephant"];
let removed = animals.splice(2, 1); // remove 1 element at index 2
console.log(removed); // ["rat"]
console.log(animals); // ["dog", "cat", "lion", "elephant"]

// Add new elements
animals.splice(2, 0, "tiger", "monkey");
console.log(animals); // ["dog", "cat", "tiger", "monkey", "lion", "elephant"]
```

üí° **Key difference:**

* `slice()` ‚Üí original array **unchanged**
* `splice()` ‚Üí original array **modified**
